You are viewing Taylor's backend index.html, to use this backend service, you can access the couchDB database by using:

  • Curl command in terminal
  • Axios api of React
  • Or, Postman app (for testing and demonstration)

The backend implemented RESTful design, which provides a standard access method providing a great deal of flexibility for future development. Implemented method including:

  • get(_id): used for retrieving a single document by document id
  • post(_obj): used for create a document in the database
  • put(_id,_obj): used for update a document by id
  • delete(_id): used for detele a document by id

Moreover, these methods are developed to:

  • view(_name): used to lookup an existing view in database
  • query(_query): used to query the database with a query json, and receive result

To access using curl command:

get(_id): 
curl -X GET http://database_address:port/<database_name>/<id>

post(_obj): 
curl -X POST -H "Content-Type: application/json" \
-d '{"value": “something”}' \ 
http://database_address:port/<database_name>

put(_id,_obj): 
curl -X POST -H "Content-Type: application/json" \
        -d '{"value": “something”}' \ 
http://database_address:port/<database_name>/<id>

delete(_id): 
curl -X DELETE http://database_address:port/<database_name>/<id>

To query database:

curl -X POST -H "Content-Type: application/json" \
         -d '{"query": {"selector": {"language": {"$eq": "jp"}}}}' \
         http://database_address:port/<database_name>/query

To retrieve view:

curl -X GET http://database_address:port/<database_name>/view/<doc_name>/<view_name>

You should get a gist of how to access the database by now, no matter using Axios api, or Postman app. If any question, ask ask ask ask :P
